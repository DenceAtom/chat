# Face Filters

This directory contains images for face filters used in the video chat application.

The following filter images should be placed here:
- dog.png
- cat.png
- bunny.png
- glasses.png
- hat.png
- crown.png
- mustache.png
- alien.png
- zombie.png
- heart.png
- fire.png

Each image should ideally have a transparent background (PNG format) and be sized appropriately for overlaying on a face.
